# Salat Times

Salat (Namaz) timetable for any location around the world.

Displays Salat (Namaz) timetable on your site for any location around the world, based on a variety of calculation methods currently used in Muslim communities.

## Usage

- Install and activate the plugin.
- Navigate to: Settings > Salat Times.
- Use widget or shortcode.

## Main Features

- Easy admin panel.
- Beautiful widgets.
- Automatic/Manual time.
- Hijri and Gregorian date.
- Timetable for any location.
- Language support: English, Bengali, and others.
- Different calculation methods for different regions.

## Installation

1. Download the plugin.
2. Go to: Dashboard > Plugins > Add New > Upload.
3. Upload, install, and activate the plugin.
4. Done.
5. Use widget: Daily Salat Times.

## Frequently Asked Questions

### How does it work?

- Widget: Daily Salat Times
- Shortcode: [daily_salat_times]
- PHP code:

```php
<?php echo do_shortcode('[daily_salat_times]'); ?>
```

## Upgrade Notice

After upgrading to version 2.0 from version 1.3 or older, click on Restore Default Settings on the Salat Times Settings page. Otherwise, the widget may not work properly.
